# vue3-cart

使用Vue3 與 Tailwind CSS 完成簡易購物車

## Demo

See [Demo](https://cathywu2017.github.io/vue3-cart/dist/).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
